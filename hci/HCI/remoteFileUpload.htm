Remote File Upload

Icons can be uploaded from or linked to a remote source to be used in KML generation

Assumptions/Prerequirements

You can upload icons from the Main page, or from within a Modify Connection page.

Instructions For adding an icon to the library from a local source

  1. On the main page click Upload Icon

    Or click Upload Icon on the Modify Connection page
  2. If desired check next to Fetch
  3. Type or paste a link into the text field next to Save
  4. Click Save

Warnings

Only bitmap (bmp), portable network graphics (png), x-png, jpeg (jpg), and tagged image file format (tiff) file types are allowed.

Only icons of size 128 by 128 or smaller are allowed.

Notes

If an error occurs, check the above two warning conditions are met and that the link is valid (copy and paste it into an Internet Browser and verify that the image/link is valid).

The fetch box stores the image pasted into the link on the server. If fetch is not checked a link to the image will be stored and visited when needed.
