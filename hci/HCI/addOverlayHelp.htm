Add Overlay Color

Overlay Colors can be added to connections to be displayed by the KML Generator.

Assumptions/Prerequisites

In order to add an icon overlay color, you must first:

  1. Create a connection

Instructions For Adding an Overlay Clor from a Connection

  1. In the Overlay Colors box click the Add Overlay button
  2. In the Overlay Color Library popup box click the white box in the center under Select Color:
  3. Click one of the color boxes or a color on the color palate to select the desired overlay color
  4. Click the Submit button on the Overlay Color Library popup box to apply the Overlay Color to the connection

Warnings

A connection may have as many overlay colors as the user desires. However, no individual color may be used more than once per connection.
