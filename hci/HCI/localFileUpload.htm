Local File Upload

Icons can be uploaded from a local source to be used in KML generation

Assumptions/Prerequisites

You can upload icons from the Main page, or from within a Modify Connection page.

Instructions For adding an icon to the library from a local source

  1. On the main page click Upload Icon

    Or click Upload Icon on the Modify Connection page
  2. click Upload Icon Locally
  3. click Browse
  4. click Submit after an image has been selected in the box that appears after clicking Browse

Warnings

Only bitmap (bmp), portable network graphics (png), x-png, jpeg (jpg), and tagged image file format (tiff) file types are allowed.

Only icons of size 128 by 128 or smaller are allowed.

Notes

If an error occurs, check the above two warning conditions are met.
