Remove Overlay Color

Overlay Colors can be removed from connections when they are no longer required.

Assumptions/Prerequisites

In order to remove an icon overlay color, you must first:

  1. Create a connection
  2. Add an overlay color to the connection

Instructions For Removing an Overlay Clor from a Connection

  1. In the Overlay Colors box click the Remove Overlay button
  2. In the Remove Overlay Color popup box click the color box of the overlay color that you want to remove

Warnings

When an Overlay Color is deleted from a connection the Overlay Color and all conditions associated with that Overlay Color are deleted.
