Preview KML Help Page

Preview KML allows you to view the KML file for the current connection that is
being created. It has all of the main features of Google Earth, excluding the
navigation bar on the left side of Google Earth.

Assumptions/Prerequirements

A new connection or current connection must have been created.

Instructions

  1. Log into ODBC2KML.
  2. Find the connection you would like to edit.
  3. Click edit.
  4. Click save and edit.
  5. Adjust the connection information, as needed.
  6. Click Preview KML.
  7. KML file will populate through Google Earth plugin in a popup.

OR

  1. Log into ODBC2KML.
  2. Click new connection.
  3. Enter database information and click save.
  4. Fill out all required fields.
  5. Fill in any additional description, icon, or overlay information.
  6. Click Preview KML.
  7. KML file will populate through Google Earth plugin in a popup.
Sample Image width of 875px

Warnings

Preview KML will not work if there is no internet access.
The database being connection to may be on a local network
and thus still work, but if the Preview KML function cannot
access the internet, it will not be able to pull the information
needed from Google to create the Google Earth globe. Without the
globe, the KML file cannot be previewed.
