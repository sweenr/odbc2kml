Add Icon

Icons can be added to connections to be displayed by the KML Generator.

Assumptions/Prerequirements

In order to add an icon, you must first:

  1. Create a connection
  2. Upload an icon into the Icon Library

Instructions For Adding an Icon from a Connection

  1. In the Icons box click the Add Icons button
  2. In the Icon Library popup box click the icon that you want to add to the connection

Warnings

A connection may have as many icons as the user desires. However, no individual icon may be used more than once per connection.
