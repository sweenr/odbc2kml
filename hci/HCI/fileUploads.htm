File Uploads

The file upload tool can be used to add user icons to the KML files generated by the ODBC2KML.

Assumptions/Prerequisites

None

Notes

Icons can be uploaded from a local source (the user's computer) or from a remote source. The fetch option is used to either download the icon to the server for future use if fetch is checked, or to save a link that will be used for the creation of KML's without saving the icon to the server if the fetch box is unchecked.
