Update Help Page

When you update your connection information, the program will try and determine
if the current information used in the connection is still valid. In other words,
if the mapping information or icon or overlay condition no longer exists in the
new database, all of the inaccurate information will be purged from that connection.
This has to happen in order to ensure that the state of that connection is valid.

Assumptions/Prerequirements

There must be a connection to update.

Instructions

  1. Log into ODBC2KML.
  2. Find the connection you would like to edit.
  3. Click edit.
  4. Click save and edit.
  5. Adjust the connection information, as needed.
  6. Click update.
Sample Image width of 875px

Warnings

If the mapping or any icon or overlay condition is no longer valid,
the description will automatically be removed. If the database information
entered is incorrect, an error will be presented and the information will remain the same.
