Description Help

The description box allows you to add a description to the KML placemarks generated.

Assumptions/Prerequisites

In order to map latitude and longitude for a connection, you must first:

  1. Create a connection
  2. Edit a connection

Instructions

Any plain text in the description box will be used in all descriptions. To insert a field value that changes with each row:

  1. Click the Field Value button above the description box.
  2. Select the field that contains the value you want to insert into the description.
  3. Click Insert Field.

To insert the currently mapped table name, click Insert Table button.

        

To insert a URL into the description:

  1. Click the Insert Link button above the description box.
  2. Enter the title for the URL, as well as the address for the link.
  3. Click Insert Link.

Warnings

If there is an error with the tags in your description, you will recieve an error message when you try to save the connection. Verify that all of your opening tags have closing tags and that there is only one sub-tag of each type in the field and url tags.

Notes

You can also type the tags in manually:

Table Name:[TBL/]

Field Value:[FIELD][TBL]table_name[/TBL][COL]column_name[/COL][/FIELD]

Link:[URL][TITLE]url_title[/TITLE]url_location[/URL]

Tags are not case-sensitive, i.e. [title] is the same as [TITLE].
