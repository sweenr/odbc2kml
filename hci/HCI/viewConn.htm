View Connection

Viewing a connection is a read-only way to view the selected connection. This allows you to see the connection without editing it.

Assumptions/Prerequisites

In order to delete a connection, you must first:

  1. Create a connection

Instructions

  1. Click the "Open Connection" icon on main page next to the connection you wish to view.

Warnings

  • If the system can not connect to the ODBC database an error will be presented to you.

Notes

  • Viewing a connection is read-only and can not be edited.
