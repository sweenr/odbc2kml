Edit Connection

The main page allows you to change a connection's information before changing any of the connection's details.
After verifying the connection information on the Main.aspx page you will be redirected to the ConnDetails.aspx page.

Assumptions/Prerequisites

In order to edit a connection the connection must already exist.

Instructions

  1. Click the edit button next to the connection you want to edit:
  2. Change any information and verify that all the required information is correct.
  3. If you want to just save this information go to 3.1, otherwise if you want to edit go to 3.2
    1. Click Save


      then Yes




    2. Click Save and Edit


      then Yes


      You will then be redirected to the ConnDetails.aspx page to change connection settings.

Warnings

When you select Yes on the warning dialog the system will sanitize the connection.
Sanitizing the connection does the following:

  • Validates the current connection information
  • Checks the current mapping to ensure that the proper tables and fields exist in the database. If not, the connection will not save
  • Ensures that the description field does not contain any invalid tags. If it does the description will be cleared
  • Any conditions that use table name/ field name combinations that no longer exist in the remote database will be deleted
