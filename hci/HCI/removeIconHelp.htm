Remove Icon

Icons can be removed from connections when they are no longer required.

Assumptions/Prerequirements

In order to remove an icon, you must first:

  1. Create a connection
  2. Add an icon to the connection

Instructions For Removing an Icon from a Connection

  1. In the Icons box click the Remove Icons button
  2. In the Remove Icons popup box click the icon that you want to remove

Warnings

When an Icon is deleted from a connection the Icon and all conditions associated with that icon are deleted.
