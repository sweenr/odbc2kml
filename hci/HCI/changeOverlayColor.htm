Change Overlay Color

You can change an Overlay's color on the ConnDetails.aspx page.
This change will only propagate to newly generated KML files.

Assumptions/Prerequisites

In order to change an overlay color you must do the following first:

  1. Create a Connection
  2. Edit a Connection
  3. An overlay must already exist. For help with that please see Add Overlay Help

Instructions

  1. Click the color box for the overlay you want to change:

  2. Select the new color:

Warnings

You must save the connection for the change to be saved.

Notes

The new color cannot be the same as any other color in that connection.
